## SNS 데이터를 이용한 여행지 분석 프로젝트

1. 인스타그램의 해시태그 데이터를 크롤링(수집)한다.
2. 수집한 해시태그 데이터를 전처리한다.
3. 공공 데이터 포털에서 내려받은 전국 날짜별 날씨 데이터셋과 수집한 인스타그램 데이터를 비교한다.
4. 날씨별 방문 빈도(해시태그된 갯수), 같이 태그된 키워드의 순위 데이터를 분석한다.
